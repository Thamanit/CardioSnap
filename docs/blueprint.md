# **App Name**: CardioCap

## Core Features:

- Nailfold Image Upload: Allow users to upload nailfold images for analysis.
- ECG Data Entry: Enable manual entry of 3-Lead ECG data (Lead I, Lead II, Lead III) in mV.
- Oxygen Saturation Input: Allow entry of PPG (Oxygen Saturation) data as SpO2 percentage.
- Risk Assessment: Generative AI tool assesses the entered data, plus uploaded image if present, and generates a risk factor analysis.

## Style Guidelines:

- Primary color: Deep red (#B71C1C) to reflect the cardiovascular theme.
- Background color: Light gray (#F0F0F0) to provide a clean and clinical feel.
- Accent color: Teal (#008080) for interactive elements and call-to-action buttons.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, machined, objective, neutral look that is suitable for both headlines and body text.
- Use heart-related icons with a modern and minimalist design.
- Clean and organized layout with clear sections for each type of data entry.
- Subtle transitions for loading data and displaying results.
